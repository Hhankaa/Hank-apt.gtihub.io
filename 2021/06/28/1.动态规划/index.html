动态规划

动态规划，包含3个重要的概念： 1. 最优子结构 2. 边界3. 状态方程

思考问题： 如何优化动态规划问题？如果dp[i] 只和dp[i-1]有关，那么可以使用【滚动变量】的方式将代码进行优化。

  • 当前状态至于前两个状态有关，只需要想办法存储前两个状态就可以了。

1. 53. 最大子序和

  • dp[i]：表示以 nums[i] **结尾的 连续 子数组的最大和。

  • 状态转移方程
    $$
    dp[i]=\left{
    \begin{aligned}
    dp[i-1]+nums[i] & & dp[i-1]>=0\
    nums[i] & & dp< 0
    \end{aligned}
    \right.
    $$

    $$
    dp[i] = max(dp[i-1]+nums[i],nums[i])
    $$

  • 思考初始问题： dp[0 ] = nums[0].

def maxSubArray(nums):
    if len(nums) == 0: return 0
    dp = [0 for _ in range(len(nums))]
    dp[0] = nums[0]
    for i in range(len(nums)):
        dp[i] = max(dp[i-1]+nums[i],nums[i])
    return max(dp)
def maxSubArray(nums):
    if len(nums) == 0: return 0
    pre = nums[0]
    res = 0
    for i in range(len(nums)):
        pre = max(pre+nums[i],nums[i])
        res = max(res,pre)
    return res

3. 121. 买卖股票的最佳时机

买一次，卖一次，max-min 就可以得到最大的利润，那就要得到这个最小值，遍历整个list，每次都与当前的最小值相减，得到当前的最大值即可。

  def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        if n < 0 : return 0
        dp = [0]*n
        minPrice = prices[0]

        for i in range(n):
            minPrice = min(prices[i],minPrice)
            dp[i] = max(dp[i-1],prices[i]-minPrice)
        return max(dp)

因为dp[i] 只和 dp[i-1]有关，所以可以进行状态压缩

    def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        if n < 0 : return 0
        minPrice = prices[0]
        maxProfit = 0
        for i in range(n):
            minPrice = min(prices[i],minPrice)
            maxProfit = max(maxProfit,prices[i]-minPrice)
        return maxProfit
